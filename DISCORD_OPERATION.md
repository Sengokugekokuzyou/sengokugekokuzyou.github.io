# 公式Discord 稼働メモ

対象サーバー: 戦国下剋上BEATS 公式  
招待URL: https://discord.gg/v37VSgrVHC

## 稼働方針

- 公開チャンネルへの投稿は代表者確認後に実施する。
- GitHubのpush通知は非公開チャンネル `自動通知` で受ける。
- Webhook URL、トークン、APIキーはドキュメントに記載しない。
- 不具合報告は、作品名、端末、ブラウザ、発生手順、スクリーンショット有無を確認する。

## 公開チャンネル

- `更新情報`: 作品更新、HP公開、ゲーム公開、重要なお知らせ。
- `感想と要望`: プレイヤーや視聴者の感想、改善要望。
- `不具合報告`: ゲームやHPの不具合報告。

## 非公開チャンネル

- `開発報告`: 制作進捗、次作業、検証結果。
- `公開前確認`: 告知文、ZIP、公開リンクの最終確認。
- `自動通知`: GitHub通知などの自動ログ。
- `代表者メモ`: 代表者の判断、保留事項、重要メモ。

## 初回投稿案: 不具合報告

不具合報告は、できる範囲で以下を添えてください。

- 作品名
- 端末: PC / スマホ / タブレット
- ブラウザ: Chrome / Edge / Safari など
- 何をしたら起きたか
- スクリーンショットの有無

いただいた報告は、開発側で確認して優先度をつけます。

## 次にやること

1. `更新情報` に初回投稿を出す。
2. `不具合報告` に報告テンプレートを固定または投稿する。
3. GitHub Pages公開後、公式HPリンクを `更新情報` に投稿する。
4. 広告収益化の審査が進んだら、HPとゲーム側の実装方針を `開発報告` に残す。
